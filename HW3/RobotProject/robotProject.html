Create a new project to start writing a program

Start a project through File > New Project to write a program for the Base Model of Robot Educator with color sensor (down) and ultrasonic sensor attached (see Picture 1).

Write a program to make the robot to behave similarly to the automated guided robot in the previous video you watched i.e. move along the black line, stop whenever there is an obstacle ahead and continue moving once the obstacle is removed. The robot model will be tested on the LEGO Mindstorms NXT 2.0 Test Pad.
